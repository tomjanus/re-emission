Installation Guidelines
=======================

.. _ReEmission: https://github.com/tomjanus/reemission

Installation as a Python Package - Option 1
-------------------------------------------

ReEmission_ is an application written in Python and has to be installed just like any Python package. The installation procedure is described in detail in :doc:`install_package`.

Installation using Docker - Option 2
------------------------------------

We also provide an alternative installation option from a Docker image. You can find more information in :ref:`What is Docker` and subseqent chapters - :doc:`building_image` and :doc:`using_image`.

.. note::
   Installing ReEmission_ with Docker requires prior installation of ``Docker Desktop`` or Docker Engine (Linux only) - see :doc:`building_image` and :doc:`using_image` for details.
